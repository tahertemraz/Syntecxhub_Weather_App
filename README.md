# 🌦️ Weather App (React)

A modern weather forecasting web application built with **React** that fetches and displays real-time weather data using the Open-Meteo API.

---

## 🚀 Live Demo

👉 [Open Live Demo](https://syntecxhub-weather-app-seven.vercel.app/)



---

## ✨ Features

- 🔍 Search for any city worldwide  
- 🌡️ Real-time temperature display  
- ☁️ Weather condition (clear, cloudy, rain, etc.)  
- 💨 Wind speed information  
- ⏳ Loading state for better user experience  
- ⚠️ Error handling for invalid or missing cities  

---

## 🛠️ Tech Stack

- React (Hooks, useState)
- JavaScript (ES6+)
- Open-Meteo API
- CSS
- Vite

---

## ⚙️ How It Works

1. User enters a city name  
2. The app fetches location data (latitude & longitude) using the Open-Meteo Geocoding API  
3. It then requests real-time weather data using these coordinates  
4. The app displays temperature, weather condition, and wind speed  
5. Loading and error states are handled for a smooth user experience  

---

## 🔗 API Used

- Open-Meteo Geocoding API  
- Open-Meteo Weather API  

https://open-meteo.com/
